Brain Rot - 10 Questions Test (Static Website)
==============================================

What's included:
- index.html
- style.css
- script.js
- images/brain_cartoon.png (placeholder)
- images/brain_real.png (placeholder)
- README.txt

How to use:
1. Unzip the package.
2. Open index.html in any modern browser to test locally.
3. To publish: upload files to GitHub repository root and enable GitHub Pages, or host on Netlify/Vercel.

Customization:
- Edit QUESTIONS in script.js to change text, options, or scores.
- The site defaults to English; press AR button to switch to Arabic.

Disclaimer:
This is an informational self-check, not a medical diagnostic tool.
